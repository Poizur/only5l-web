## Krok 1: Prompty pro psaní a komunikaci

Podle testů jsou tyto prompty nejjednoduší na použití a přinášejí okamžité výsledky.

**Email marketing:**
```
"Napiš email pro [typ zákazníka] o [produkt/služba]. Délka: [počet slov]. Tón: [profesionální/neformální]. Zahrnuj: call-to-action na [konkrétní akce]. Vyvaruj se: [co nechceš]."
```

**Praktický příklad:** Email pro majitele e-shopů o AI copywritingu. Délka 180 slov, tón profesionální ale přátelský. Výsledek: vyšší otevírací míra než u standardních šablon.

**Přepis poznámek do struktury:**
```
"Převeď tyto poznámky do strukturovaného formátu s hlavními body, podbody a action items. Přidej deadliny kde dávají smysl: [tvoje poznámky]"
```

**Odpovědi na obtížné emaily:**
```
"Pomoz mi odpovědět na tento email. Kontext: [situace]. Email: [zkopíruj text]. Chci znít: [tón]. Hlavní cíl odpovědi: [co chceš dosáhnout]."
```

**Vytvoření personalizované zprávy:**
```
"Napiš LinkedIn zprávu pro [pozice] ve firmě [název]. Zmíň: [konkrétní detail o firmě/osobě]. Cíl: [meeting/spolupráce/informace]. Délka: max 100 slov."
```

**Korektura a zlepšení textů:**
```
"Oprav gramatiku a stylistiku tohoto textu. Zachovej původní tón. Upozorni na hlavní chyby: [tvůj text]"
```

**Příprava oficiální korespondence:**
```
"Napiš oficiální dopis pro [instituce]. Účel: [žádost/stížnost/informace]. Klíčové body: [seznam]. Formát: český úřední styl."
```

**Konkrétní příklad z praxe:**
```
"Zákazník si stěžuje na zpoždění zásilky PPL o 3 dny během vánočních svátků. Chci se omluvit, nabídnout 15% slevu na další nákup a udržet zákazníka. Tón: empatický ale profesionální, max 120 slov."
```

Výsledek: Z podobných případů se podle uživatelských zpráv podařilo udržet větší procento zákazníků než u standardních odpovědí.

## Krok 2: Prompty pro analýzu a research

ChatGPT dokáže zpracovat a strukturovat informace způsobem, který přehlédneš.

**Analýza konkurence:**
```
"Proveď SWOT analýzu těchto konkurentů v oboru [tvůj obor]: [seznam konkurentů]. Zaměř se na: [specifické aspekty]. Výstup ve formátu tabulky."
```

**Praktické použití:** Analýza AI nástrojů pro copywriting (<ToolLink slug="jasper">Jasper</ToolLink>, <ToolLink slug="copyai">Copy.ai</ToolLink>, <ToolLink slug="writesonic">Writesonic</ToolLink>, Rytr, QuillBot). ChatGPT identifikuje klíčové rozdíly v ceně, funkcích a API integraci.

**Výzkum tématu:**
```
"Proveď research na téma [téma]. Potřebuji: 5 hlavních trendů, 3 největší problémy v oboru, přehled vývoje, 5 odborníků na sledování. Formát: bullet points."
```

**Analýza pro/proti:**
```
"Vytvoř objektivní analýzu pro a proti rozhodnutí: [tvé rozhodnutí]. Kontext: [situace]. Ohodnoť každý bod 1-10 podle důležitosti."
```

**Sumarizace dlouhého textu:**
```
"Sumarizuj tento text do [počet] hlavních bodů. Zaměř se na: actionable insights. Ignoruj: obecné informace. Text: [zkopíruj text]"
```

**Analýza sentimentu:**
```
"Analyzuj sentiment těchto customer reviews: [recenze]. Roztřiď na pozitivní/negativní/neutrální. Identifikuj hlavní problémy a pochvaly."
```

**Identifikace trendů:**
```
"Z těchto dat identifikuj hlavní trendy: [data/informace]. Vysvětli důsledky pro [tvůj obor]. Navrhni action steps pro využití."
```

**Benchmarking:**
```
"Porovnej naše výkony s konkurencí. Naše čísla: [tvá data]. Konkurence: [jejich data]. Metriky: [co porovnávat]. Kde zaostáváme a proč?"
```

**Český příklad - analýza trhu:**
```
"Analyzuj český e-commerce trh v letošním roce. Zaměř se na: růst segmentů, hlavní hráče (Alza, Mall, Zásilkovna), dopady EU Digital Services Act, příležitosti pro malé e-shopy."
```

Výsledek: Identifikace klíčových trendů včetně růstu udržitelných produktů a příležitostí v hyperlokalních dodávkách.

## Krok 3: Prompty pro plánování a organizaci

Nejefektivnější kategorie pro ušetření času v každodenní práci.

**Plánování projektu:**
```
"Vytvoř projektový plán pro: [název projektu]. Časový rámec: [doba]. Tým: [počet lidí, role]. Rozpočet: [orientační]. Výstup: timeline s milníky, rizika, dependencies."
```

**Praktické použití:** Plánování launch online kurzu s týmem několika lidí za určité období. ChatGPT vytvoří timeline s konkrétními úkoly, identifikuje rizika a navrh backup plány.

**Optimalizace denního rozvrhu:**
```
"Optimalizuj můj denní rozvrh. Aktuální rozvrh: [popis dne]. Prioritní úkoly: [seznam]. Omezení: [časové/energetické]. Optimalizuj pro: [produktivitu/work-life balance]."
```

**Plán na dosažení cíle:**
```
"Vytvoř step-by-step plán na dosažení: [tvůj cíl]. Současná situace: [kde jsi teď]. Časový rámec: [deadline]. Rozdel do týdenních milestone."
```

**Delegování úkolů:**
```
"Pomoz mi rozdělit tyto úkoly mezi tým: [seznam úkolů]. Členové týmu: [jména + skills]. Kritéria: [deadline/priorita/složitost]. Výstup: přiřazení + timeline."
```

**Plánování meetingů:**
```
"Vytvoř agendu pro meeting o [téma]. Účastníci: [role lidí]. Délka: [čas]. Cíl: [co chceme rozhodnout/prodiskutovat]. Zahrnuj časy pro jednotlivé body."
```

**Risk management:**
```
"Identifikuj potenciální rizika projektu: [popis projektu]. Ohodnoť pravděpodobnost a dopad 1-10. Navrhni mitigační opatření pro top 5 rizik."
```

**Backlog management:**
```
"Prioritizuj tyto úkoly podle MoSCoW metody: [seznam úkolů]. Kontext: [deadline/resources/business cíle]. Zdůvodni prioritizaci."
```

**Praktický příklad - osobní používání:**
```
"Naplánuj dovolenou po Čechách pro rodinu s dětmi. Budget: 30.000 Kč. Zájmy: příroda, hrady, aktivní odpočinek. Start a konec: Praha."
```

Výsledek: Optimalizovaná trasa s konkrétními místy, ubytováním v daném rozpočtu a aktivitami vhodných pro rodinu.

## Krok 4: Prompty pro kreativitu a brainstorming

ChatGPT kombinuje nápady neobvyklým způsobem, bez tradičních předsudků.

**Generování nápadů:**
```
"Vygeneruj nápady na [téma]. Kontext: [tvoje situace]. Cílová skupina: [popis]. Kritéria: [realizovatelnost/originalita/rozpočet]. Seřaď podle potenciálu."
```

**Praktické testování:** Hledání nápadů na LinkedIn obsah pro B2B startup. ChatGPT vygeneruje několik originálních přístupů s větším engagementem než běžný obsah.

**Řešení problémů:**
```
"Problém: [popis problému]. Omezení: [co nemůžeš změnit]. Cíl: [ideální výsledek]. Vygeneruj kreativní řešení včetně nekonvenčních přístupů."
```

**Inovace produktu:**
```
"Produkt: [popis]. Současní uživatelé říkají: [feedback]. Konkurence dělá: [co dělají jiní]. Navrhni funkce, které by nás odlišily."
```

**Storytelling:**
```
"Vytvoř příběh pro prezentaci o: [téma]. Audience: [popis publika]. Délka: [čas]. Zahrnuj: osobní zkušenost, data, call-to-action. Struktura: hook, conflict, resolution."
```

**Kreativní psaní:**
```
"Napiš kreativní text typu [blog post/reklama/příběh] o [téma]. Styl: [popis stylu]. Délka: [počet slov]. Zaměř se na emoce: [konkrétní emoce]."
```

**Naming a branding:**
```
"Navrhni názvy pro [typ produktu/služby]. Target skupina: [popis]. Asociace: [co má název vyvolávat]. Vyvaruj se: [co nechceš]. Zahrnuj české a anglické varianty."
```

**Gamifikace:**
```
"Navrhni gamifikační prvky pro [proces/aplikace]. Uživatelé: [popis]. Cíl: [co chceš motivovat]. Inspiruj se: [úspěšné příklady]."
```

**Český příklad - lokální podnikání:**
```
"Vymysli nápady na lokální podnikání v Brně. Kritéria: startovací investice do 300.000 Kč, využití místních zdrojů, zaměření na udržitelnost."
```

Výsledek: Konkrétní nápady s odhady viability a nákladů, priamo aplikovatelné v českém kontextu.

## Krok 5: Prompty pro vzdělávání a rozvoj

AI dokáže personalizovat učení podle tvého tempa a stylu.

**Vytvoření studijního plánu:**
```
"Chci se naučit: [skill/téma]. Současná úroveň: [beginner/intermediate/advanced]. Časová dotace: [hodiny týdně]. Cíl: [co chceš umět]. Vytvoř studijní plán."
```

**Praktický příklad:** Učení se Python pro datovou analýzu. Plán s progresivními projekty a konkrétními cíli v jednotlivých týdnech.

**Vysvětlení složitých konceptů:**
```
"Vysvětli [koncept] jako bych byl [věk/profese]. Použij analogie z [obor, který znám]. Zahrnuj praktické příklady. Otestuj mé porozumění otázkami."
```

**Příprava na prezentaci:**
```
"Téma prezentace: [téma]. Audience: [popis]. Délka: [čas]. Vytvoř: outline, klíčové pointy, anticipované otázky s odpověďmi, tips pro delivery."
```

**Feedback na práci:**
```
"Ohodnoť tuto práci: [zkopíruj text/projekt]. Kritéria: [co je důležité]. Poskytni: konkrétní pozitiva, 3 hlavní oblasti na zlepšení, actionable steps."
```

**Jazykové vzdělávání:**
```
"Vytvoř lekci [jazyk] na úrovni [A1-C2]. Téma: [gramatika/slovní zásoba]. Zahrnuj: vysvětlení, příklady, cvičení s řešením, praktické použití."
```

**Příprava na pohovor:**
```
"Připrav mě na pohovor na pozici [název pozice] ve firmě [typ firmy]. Vytvoř: typické otázky, mé odpovědi založené na [můj background], otázky pro tazatele."
```

**Microlearning:**
```
"Vytvoř 5minutovou lekci o [téma]. Cílová skupina: [popis]. Formát: [text/video script/infografika]. Zahrnuj: klíčový takeaway, praktický tip, další kroky."
```

**Český kontext - účetnictví:**
```
"Vysvětli základy účetnictví pro začátečníky v ČR. Zaměř se na: podvojné účetnictví, DPH, daň z příjmu, povinné doklady. Používej konkrétní české příklady."
```

Výsledek: Srozumitelný návod s praktickými příklady a konkrétními čísly v korunách.

## Časté chyby a jak je vyřešit

| Chyba | Proč se stává | Řešení |
|-------|--------------|--------|
| Příliš obecné prompty | "Pomoz mi s emailem" místo konkrétního zadání | Přidej kontext, čísla, konkrétní cíl |
| ChatGPT ignoruje instrukce | Prompt má příliš mnoho instrukcí najednou | Rozděl na 3-5 jasných bodů, použij číslování |
| Odpovědi nejsou praktické | Chybí real-world kontext a omezení | Přidej svou situaci a konkrétní omezeníí |
| Opakování stejných nápadů | Nezadáváš variace v promptech | Požádej o "nekonvenční" nebo "inovativní" přístupy |
| Slabé výsledky v češtině | Různá kvalita českého modelu | Pro složitější úkoly zadávej anglicky, pak přeloži |
| Ztráta kontextu v konverzaci | Dlouhá diskuse bez shrnutí | Každých 8-10 zpráv shrň dosavadní progress |
| Hallucinating faktů | AI si vymýšlí čísla a citace | Vždy ověř faktické informace z různých zdrojů |
| Generické odpovědi | Chybějí osobní detaily | Přidej kontext: tvůj background a konkrétní situaci |

## Krok 6: Pokročilé techniky promptování

Pro situace, kdy potřebuješ maximum ze ChatGPT.

**Chain of Thought prompting:**
```
"Vyřeš tento problém step-by-step. Ukaž své myšlení v každém kroku: [problém]. Před finální odpovědí zhodnoť, jestli dává smysl."
```

**Role-based prompting:**
```
"Představ si, že jsi [expert role] s dlouhodobou praxí v [obor]. Analyzuj tuto situaci z tvé perspektivy: [situace]. Jaký by byl tvůj postup?"
```

**Multi-perspective analysis:**
```
"Analyzuj [téma/problém] ze 3 úhlů: 1) Technický expert, 2) Business manažer, 3) Koncový uživatel. Každý pohled max 100 slov."
```

**Iterative improvement:**
```
"Vezmi tuto první verzi: [tvůj text/nápad]. Zlepši ji v jednotlivých iteracích - pokaždé se zaměř na jiný aspekt: 1) Jasnost, 2) Přesvědčivost, 3) Praktičnost."
```

**Few-shot prompting:**
```
"Udělej to podle těchto příkladů:
Příklad 1: [vstup] → [výstup]
Příklad 2: [vstup] → [výstup]
Teď udělej to samé pro: [tvůj vstup]"
```

**Constraint-based prompting:**
```
"Řeš [problém] s těmito omezeními: max 100 slov, bez odborných termínů, pro lidi bez technického vzdělání, musí obsahovat konkrétní příklad."
```

## FAQ

**Jak dlouhý může být prompt?**

ChatGPT zpracuje prompty do 4000 tokenů (cca 3000 slov). Delší texty rozděl na části.

**Proč ChatGPT někdy ignoruje instrukce?**

Nejčastěji kvůli přílišné složitosti promptu. Rozděl na 3-5 jasných bodů, použij číslování.

**Lze naučit ChatGPT můj styl psaní?**

V rámci jedné konverzace ano. Poskytni 2-3 ukázky svého stylu s instrukcí "Write in similar style and tone."

**Jak ověřit správnost informací?**

ChatGPT může generovat nesprávná data. Faktické informace, čísla a citace vždy ověř z nezávislých zdrojů.

**Fungují tyto prompty v jiných AI nástrojích?**

Základní struktura ano, ale každý AI má specifika. Claude preferuje delší kontexty, <ToolLink slug="gemini">Gemini</ToolLink> je lepší s aktuálními informacemi.

**Jak často aktualizovat prompty?**

Alespoň jednou za 3 měsíce. AI modely se vyvíjejí a vznikají efektivnější techniky.

**Lze používat ChatGPT pro firemní účely?**

Ano, ale pozor na citlivá data. Pro business použití zvažuj ChatGPT Enterprise.

**Jaká je kvalita českých textů?**

Pro běžné texty velmi dobrá, pro odborné texty doporučuji korekturu editorem.

**Rychlost zpracování promptů?**

Podle modelu a složitosti 2-30 sekund. Plus verze má prioritní přístup.

**Limity na počet dotazů?**

Bezplatná verze: omezené dotazy během dne. Plus: větší limit.

## Závěr

Efektivní práce s ChatGPT není o hledání magických formulací, ale o systematickém přístupu. Klíčové principy:

1. **Specifičnost vítězí** - "Napiš email" vs "Napiš follow-up email pro neplatící klient, tón diplomatický, cíl: splátkový kalendář"
2. **Struktura pomáhá** - číslované body, jasné instrukce, konkrétní očekávání
3. **Iterace je normální** - první prompt je začátek, upřesňuj podle výsledků
4. **Kontext rozhoduje** - čím více relevantních informací, tím lepší výstup

Doporučuji začít s 5 prompty z kategorií, které používáš nejčastěji. Za měsíc pravidelné práce uvidíš výrazné zlepšení kvality výstupů.

## Proč doporučujeme

ChatGPT s kvalitními prompty dokáže změnit způsob, jak zpracováváš informace, tvoříš obsah a řešíš problémy.

- **Pro začátečníky:** Začni komunikačními prompty (emaily, odpovědi zákazníkům)
- **Pro pokročilé:** Experimentuj s role-based a chain-of-thought technikami
- **Pro firmy:** Investuj čas do vytvoření knihovny firemních promptů
- **Pro kreativce:** Používej AI jako sparring partnera, ne jako náhradu kreativity

Správně použitý ChatGPT není konkurence lidské inteligence, ale její mocné rozšíření o rychlost a systematičnost.
